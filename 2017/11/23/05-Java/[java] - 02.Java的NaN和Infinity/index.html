1. 背景知识

​ Java在Double和Float类都定义了NaN和Infinity, 这是对IEEE-754定义的NaN/正无穷大/负无穷大的实现, 介绍Java的NaN/Infinity之前,先了解下IEEE-754的定义.

1.2 NaN

​ 数学运算中, 很多实数浮点数运算是非法的, 比如取负数的平方根, 这种操作的结果会导致 浮点异常(floating-point *exception) [注: 这个异常是IEEE定义的,非Java定义的Exception]. 就用 NaN(Not A Number)来表示这类未定义或者没法表示的值.

​ NaN是面向浮点数定义的(包括单精度和双精度),造成返回NaN的运算操作有:

  • 操作数中至少有一个是 NaN 的运算
  • 未定义操作
    • 下列除法运算：0/0、∞/∞、∞/−∞、−∞/∞、−∞/−∞
    • 下列乘法运算：0×∞、0×-∞
    • 下列加法运算：∞ + (−∞)、(−∞) + ∞
    • 下列减法运算：∞ - ∞、(−∞) - (−∞)
  • 产生复数结果的实数运算。例如：
    • 对负数进行开方运算
    • 对负数进行对数运算
    • 对比-1小或比+1大的数进行反正弦或反余弦运算

1.3 无穷大

​ 指数域全为1且尾数域全为0用于表征无穷大, 符号位为0表示正无穷, 为1表示负无穷. 特征:

  • sign = 0 for positive infinity, 1 for negative infinity.
  • biased exponent = all 1 bits.
  • fraction = all 0 bits.

1.4 浮点数数据域

​ IEEE-754定义了单精度和双精度浮点数数据域:

  1. float
  • +Infinity : 0x7F800000 (S=0; 指数全1; 尾数全0)
  • -Infinity : 0xFF800000 (S=1; 指数全1; 尾数全0)
  • NaN : [0x7F800001, 0x7FFFFFFF] & [0xFF8, 0xFFFFFFFF] (S=1或1; 指数全1; 尾数不全为0, NaN 是一个范围)
  • +0/-0 : 符号位0/1, 指数位和尾数位全部为0
  • 其余为常规表达范围

  1. double (略)
