#Java的字面常量值

[TOC]

1. literal

​ Java规范共给出53个保留标识符, 其中50个是关键字, 3个字面常量 true/false/null.

​ Java规范给出了不将true/false/null归类为关键字的原因: 技术层面的实现非常简单.

​ 也就是说, 这3个字面量标识符, 仅用着变量赋值, 在编译阶段结束后, 对应着3条极其简单的处理指令. 因此Java spec 为将其纳入Java语言关键字范畴. 但是这3个词也不能用作变量名出现, 可看作为Java的3个保留字.
